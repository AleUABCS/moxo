# Comandos hasta ahora
* **leer _variable_** guarda una variable con el nombre que hayas escrito en la terminal. El valor se guarda presionando enter
* **imprimir** (_variable_ / _"texto"_ ) imprime (por ahora en la consola del navegador) el texto o valor de la variable si es que ya fue guardada con **leer**
* **si** Por ahora solo imprime los operadores (= < >) y los números en consola separados para la actividad de separar los tokens
