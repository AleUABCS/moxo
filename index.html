Terminal
Cómo mostrar texto en la terminal 
Ejemplos:

imprimir variable
imprimir "texto"

Muestra texto en la terminal. Puedes imprimir el valor de una variable o texto entre comillas.

Cómo crear variables

Formas:

Ejemplos:

nombreVariable1 = leer
nombreVariable2 = 5
nombreVariable3 = texto

Con "leer": presiona Ejecutar, escribe el valor en la terminal y presiona Enter. Con valor directo, se asigna en el código.

Puedes asignarle un valor numérico o alfabético.

Las variables pueden tener el nombre que quieras, pero solo puede estar conformado por letras, números y guión bajo. Tampoco puede empezar por un número.

Cómo usar condiciones 
si (valor1 comparador valor2)
    bloque de código
fin_si

Ejecuta el bloque solo si la condición es verdadera. Los comparadores son: == (igual), > (mayor que) < (menor que) >= (mayor o igual) <= (menor o igual) != (no es igual a) 

Ejemplo:

si (edad >= 18)
    imprimir "Usted es mayor de edad"
fin_si
Este ejemplo significa: si la edad es mayor o igual a 18, entonces escribe en la terminal "Usted es mayor de edad"
Cómo repetir código varias veces 
mientras (valor1 comparador valor2)
    bloque de código
fin_mientras

Repite el bloque mientras la condición sea verdadera.

Ejemplo:

mientras (x < 10)
    x = x + 1
    imprimir x
fin_mientras
Operaciones — Asignar con cálculos

Al asignar valores puedes usar operaciones matemáticas: + (suma) - (resta)* (multiplicación) / (división)

Ejemplos:

nombreVariable = x + 1 / 2
nombreVariable = x + 10 * 3
nombreVariable = x - 3
